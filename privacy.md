# 隐私政策

**生效日期：2026年2月25日**

## 1. 引言
欢迎使用「my email扩展」（以下简称“本扩展”）。  
本扩展是一个 Chrome 浏览器扩展，通过 Google Gmail API 帮助用户更便捷地处理邮件。  
我们非常重视您的隐私，本政策将说明本扩展如何收集、使用和保护您的信息。

## 2. 我们收集的信息
- **Gmail 数据**：用户通过 OAuth 授权后，本扩展仅访问您明确授权的 Gmail 邮箱内容（例如：读取邮件列表、邮件正文、标签等）。
- **本地数据**：扩展在您的浏览器本地存储必要的设置（如主题、过滤规则），这些数据**不会**上传到任何服务器。
- **不收集**：我们不会收集您的姓名、电话、支付信息、位置等任何与 Gmail 无关的个人信息，也不会记录您的 IP 地址或浏览行为。

## 3. 我们如何使用您的信息
本扩展仅在您的浏览器内使用获取的 Gmail 数据，用于实现以下功能：
- 显示邮件列表
- 快速搜索/过滤邮件
- 执行用户手动触发的操作（如标记已读、移动标签）

**所有数据处理均在您的设备本地完成，不会发送给第三方服务器。**

## 4. 数据共享与披露
- 我们**绝不会**将您的 Gmail 数据出售、出租或分享给任何第三方。
- 仅在法律要求（如法院传票）或保护我们合法权益时，才会依法披露。

## 5. 数据存储与安全
- 您的 Gmail 授权令牌由 Google 安全存储，本扩展仅临时使用。
- 所有本地存储数据均加密保存。
- 本扩展不使用任何云服务器存储您的邮件内容。

## 6. 您的权利
- 您可以随时在 Chrome 扩展设置中**撤销 Gmail 授权**（chrome://extensions → 详细信息 → 查看网站数据 → 移除）。
- 您可以随时删除扩展，所有本地数据将自动清除。
- 如需删除 Google 侧的授权记录，请访问：https://myaccount.google.com/permissions

## 7. 儿童隐私
本扩展不面向 13 岁以下儿童提供服务，也不主动收集儿童信息。

## 8. 政策更新
我们可能会不定期更新本隐私政策，并在 GitHub 仓库中发布最新版本。
